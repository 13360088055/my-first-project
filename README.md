# 小米 + 苹果双控智能家居项目
> 目标：实现米家设备接入 Home Assistant，同时支持米家 App 与苹果 HomeKit 双平台控制。

## 🎯 项目目标
- 统一管理米家设备，避免多 App 切换
- 通过 Home Assistant 桥接，让苹果设备（iPhone/iPad/Siri）直接控制米家设备
- 搭建自定义自动化场景（如“开门自动开灯”“回家自动开空调”）
- 所有配置云端备份，设备故障可快速恢复

## 🛠️ 设备清单
| 设备类型 | 型号/品牌 | 用途 |
| :--- | :--- | :--- |
| 中枢主机 | 树莓派 4B / 迷你主机 | 运行 Home Assistant |
| 智能插座 | 小米智能插座 | 控制普通家电电源 |
| 智能灯 | 小米智能灯 | 灯光控制与场景联动 |
| 传感器 | 人体传感器/门窗传感器 | 自动化触发条件 |

## 📂 仓库文件说明
- `/config/`：Home Assistant 核心配置文件（`configuration.yaml`、自动化规则等）
- `/scripts/`：自定义 Python 脚本（设备控制、数据同步等）
- `/docs/`：项目搭建教程、设备接线图、排错笔记
